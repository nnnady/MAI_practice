Практика на предприятии "Модуль"

Задание:
1. В классе Apupp реализуйте поддержку контекстного менеджера. Перед входом в контекстный менеджер должно устанавливаться соединение, при выходе разрываться. Используйте соответствующие APUPP команды.
2. Уберите из конструктора класса Apupp установку соединения с АПУ-ЦВМ12Р отправкой команды. Удалите деструктор класса Apupp.
3. Создайте свой скрипт на python, со следующим функционалом:
   Установите протокол взаимодействия с АПУ-ЦВМ12Р APUPP – для этого импортируйте модуль set_apupp и вызовите его ф-цию run();
   Создайте контекстный менеджер класса Apupp;
   Внутри контекстного менеджера установите значение КПУ для ВК 1 равное 0000. Попробуйте устанавливать другие значения, между установкой сделайте паузы, чтобы на пульте можно было увидеть переключение светодиодов КПУ.
   Примечание: ВК – вычислительный канал ЦВМ12Р, выбор ВК осуществляется аргументом svk.
5. Запустите скрипт на реальном стенде;
6. Добавьте в скрипт проверку установки КПУ командой считывания значения КПУ. Снова запустите на реальном стенде.
